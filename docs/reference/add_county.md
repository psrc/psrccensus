# Add County Name to PUMS API Result

Helper to
[`get_psrc_pums`](https://psrc.github.io/psrccensus/reference/get_psrc_pums.md)
function. Attaches county name.

## Usage

``` r
add_county(dt)
```

## Arguments

- dt:

  PSRC data.table

## Value

PSRC data.table with county names

# Add regional totals

Helper function for
[`get_acs_single`](https://psrc.github.io/psrccensus/reference/get_acs_single.md)
to add regional totals to county estimates

## Usage

``` r
add_regional_acs(dt)
```

## Arguments

- dt:

  data.table with Census API result

## Value

dt with regional totals

# Read PUMS csv

Helper to the
[`pums_ftp_gofer`](https://psrc.github.io/psrccensus/reference/pums_ftp_gofer.md)
function

## Usage

``` r
read_pums(target_file, dyear)
```

## Arguments

- target_file:

  either .csv from ftp site, or compressed .gz file on server

- dyear:

  The data year

## Value

unzipped table

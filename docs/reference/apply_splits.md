# Internal helper to apply splits in \`use_geography_splits\` or \`census_to_customgeo\`

Internal helper to apply splits in \`use_geography_splits\` or
\`census_to_customgeo\`

## Usage

``` r
apply_splits(df, rosetta)
```

## Arguments

- df:

  acs or decennial dataset returned from psrccensus

- rosetta:

  splits file either from Elmer TVF or calculated from parcel data

## Value

dataframe summarized to planning geography

# Helper function to [`acs_varsearch`](https://psrc.github.io/psrccensus/reference/acs_varsearch.md) and [`decennial_varsearch`](https://psrc.github.io/psrccensus/reference/decennial_varsearch.md)

Helper function to
[`acs_varsearch`](https://psrc.github.io/psrccensus/reference/acs_varsearch.md)
and
[`decennial_varsearch`](https://psrc.github.io/psrccensus/reference/decennial_varsearch.md)

## Usage

``` r
pull_varlist(survey)
```

## Arguments

- survey:

  tidycensus code for target survey

## Value

data.table of filtered variable codes and attributes

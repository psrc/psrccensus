# Get single ACS table

Core function to get ACS data for any geography

## Usage

``` r
get_acs_single(params)
```

## Arguments

- params:

  list of parameter combinations from
  [`create_param_grid`](https://psrc.github.io/psrccensus/reference/create_param_grid.md)

## Value

dt formatted ACS data for a single geography and survey

# Helper to verify custom geometry input meets criteria for geographic conversion

Helper to verify custom geometry input meets criteria for geographic
conversion

## Usage

``` r
verify_custom_geo(custom_geo, custom_geo_var)
```

## Arguments

- custom_geo:

  custom sf file with intended geography/geometry

- custom_geo_var:

  grouping field from custom geography file, i.e. geography label

## Value

verified sf object

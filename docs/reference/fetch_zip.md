# Fetch ZIP

Helper to the
[`pums_ftp_gofer`](https://psrc.github.io/psrccensus/reference/pums_ftp_gofer.md)
function

## Usage

``` r
fetch_zip(zip_filepath, target_file, dyear)
```

## Arguments

- zip_filepath:

  ftp URL location

- target_file:

  .zip archive file to read

- dyear:

  The data year

## Value

unzipped table

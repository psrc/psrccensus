# Get single Decennial Census table

Core function to get Decennial Census data for any geography

## Usage

``` r
get_decennial_single(params)
```

## Arguments

- params:

  list of parameter combinations from
  [`create_param_grid`](https://psrc.github.io/psrccensus/reference/create_param_grid.md)

## Value

dt formatted Decennial Census data for a single geography and year
